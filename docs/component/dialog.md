# Dialog 对话框

::: tip
Dialog 基于 Popup 开发，通过 [透传 Attributes](https://cn.vuejs.org/guide/components/attrs.html) 可以使用 Popup 的属性
:::

## Dialog API

### Dialog Attributes

部分属性继承 Popup 组件，详情请参考文档

| 名称      |       说明       |   类型 | 默认值 |
| --------- | :--------------: | -----: | ------ |
| title     |       标题       | string | ''     |
| showClose | 是否显示关闭按钮 | string | false  |
